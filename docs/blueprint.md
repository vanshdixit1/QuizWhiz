# **App Name**: QuizWhiz

## Core Features:

- Free Quizzes: Access to free quizzes on subjects like geography, general knowledge, history, and English.
- Quiz Timer: Option to turn the timer on or off, and set the timer duration (15/30 seconds).
- Premium Tier Access: User can purchase a premium subscription with different payment options and pricing in rupees.
- Quiz Generation from Topic Name: Premium users can generate quizzes by providing a topic name, using an AI tool.
- Quiz Generation from PDF Upload: Premium users can upload a PDF file and generate a quiz from its content using an AI tool.
- Chatbot Support: An AI-powered chatbot to assist users with their queries and provide guidance, acting as a helpful tool.
- User Dashboard: A personalized dashboard where users can view their attempted quizzes and scores.
- User Accounts: Allow the user to login/create account

## Style Guidelines:

- Primary color: Dark Lavender (#947BD3) to create a modern, sleek impression, leaning into associations of creativity and intelligence.
- Background color: Pale Lavender (#EEEAF6), a very light, desaturated tint of the primary to maintain harmony but also emphasize the content areas.
- Accent color: Light Reddish-Pink (#D37B94) to complement the primary, to guide users to calls to action without overpowering the design.
- Body and headline font: 'Inter', a sans-serif font offering a clean and modern feel suitable for various text sizes.
- Use simple, clear icons to represent different quiz categories and features. Line icons are preferable for a clean design.
- A clean, organized layout with a clear separation between the header, main content, and footer. Use a grid system for consistent alignment.
- Subtle transitions and animations when navigating between quizzes and viewing scores.
